    # 全局变量与局部变量

    全局变量声明在函数外部，在程序整个运行周期都有效，函数内部可以访问全局变量。

    局部变量声明在函数内部，函数运行期间有效，外部无法访问。

    函数中变量的查找顺序

    1. 先在函数内部查找
    2. 找不到再从函数外部查找，直到找到全局
    var x = 10

func foo()  {
  y := 100

  fmt.Println(x, y) // 10 100
}

func main()  {
  foo()
}

    # 高阶函数

    函数也可以作为一个类型，充当函数的参数或者返回值。可以使用type关键字来定义。

    type foo func(int, int) int
    1

    上述就定义了一个foo类型，这个类型的函数接收两个int类型的参数且返回一个int类型的值。

    # 函数作为参数的类型

    // 这里的 x 表示一个不接收参数且返回值为int类型的函数，
func foo(x func() int) {
  fmt.Println(x())

    # 闭包

    所谓闭包，形式上就是一个函数的返回值为另一个函数。目的是向外暴露函数作用域内部的元素。

    可以说闭包是一个函数携带了他原本所在的环境(作用域中的变量等)。

    func foo() func() {
	x := 10

	return func() {
		x++
		fmt.Println(x)
	}
}
func main() {
	bar := foo()

	bar() // 11
	bar() // 12
}

    panic、recover 错误处理

    # panic和recover

    在go(1.14)中目前是没有异常机制的，但是可以用panic/recover模式模拟

    func f1() {
  fmt.Println(1)
}

func f2() {
  // 会在panic之前释放数据库连接
  defer func() {
    // 获取出错信息，尝试恢复panic的状态
    // 恢复后，下面的 f3 函数可以继续执行
    err := recover()
    fmt.Println(err)
    fmt.Println("释放数据库连接")
  }()
  panic("出现错误！")
  fmt.Println(2)
}

func f3() {
  fmt.Println(3)
}

func main() {
  f1()
  f2()
  f3()
}
