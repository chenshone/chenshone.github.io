    # for range

    for range可以遍历数组、切片、字符串、map和channel。

    返回值

    1. 数组、切片、字符串返回索引和值
    2. map返回键和值
    3. channel只返回通道内的值
    func foo() {
  s := "hello"

  for i, v := range s {
    fmt.Println(i, v)
  }
}

    # switch

    用于简化if操作

    func foo() {
  i := 1
  switch i {
  case 1:
    fmt.Println(1)
  // 一个分支可以有多个值
  case 2, 3:
    fmt.Println(2)
  // 分支也可以使用表达式
  case i > 100
    fmt.Println(100)
  default:
    fmt.Println(4)
  }
}

    # goto(不建议用)

    跟c语言一样，可以定义一个标签，然后使用goto快速跳到指定标签位置

    func foo() {
  for i := 0; i < 10; i++ {
    if i == 2 {
      goto bar
    }
    fmt.Println(i)
  }
  bar:
    fmt.Println("end")
}
