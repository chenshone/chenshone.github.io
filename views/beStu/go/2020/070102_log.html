    go学习(十七) 日志库

    CHENSHONE 2020/7/1 golang

    日志是一个非常重要的功能，在go中内置的 log (opens new window) 包实现了简单的日志服务。

    package main

import (
	"log"
)

func main() {
	log.Println("这是一条很普通的日志。")
	v := "很普通的"
	log.Printf("这是一条%s日志。\n", v)
	log.Fatalln("这是一条会触发fatal的日志。")
	log.Panicln("这是一条会触发panic的日志。")
}

    logger会打印每条日志信息的日期、时间，默认输出到系统的标准错误。Fatal系列函数会在写入日志信息后调用os.Exit(1)。Panic系列函数会在写入日志信息后panic。

    内置的log包功能不是很全，一般还是使用第三方的库，如logrus (opens new window)、zap (opens new window)等
