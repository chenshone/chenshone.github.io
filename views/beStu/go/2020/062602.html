    go学习(二) 基本数据类型

    CHENSHONE 2020/6/26 golang

    go的基本数据类型包含整型、浮点型、布尔型和字符串

    # 整型

    类型 描述
    uint8(byte型) 无符号8位整数
    uint16 无符号16位整数
    uint32 无符号32位整数
    uint64 无符号64位整数
    int8 有符号8位整数
    int16(short型) 有符号16位整数
    int32 有符号32位整数
    int64(long型) 有符号64位整数

    # 特殊整型

    类型 描述
    unit 32位OS上就是uint32,64位OS上就是uint64
    int 32位OS上就是int32,64位OS上就是int64
    uintptr 无符号整型，用于存放一个指针

    用来获取对象的长度的len()函数返回的值可根据不同平台的字节长度进行变化。

    # 八进制和十六进制

    在go中无法直接定义二进制数，对于八进制和十六进制可以如下表示

    // 八进制 以0开头
var a int = 077
// 十六进制 以0x开头
var b int = 0xff

    # 浮点型

    go中支持两种浮点数：float32和float64

    默认go中的小数都是float64类型

    // 显示声明float32类型
f1 := float32(1.2345)

// 科学计数法
f2 := 3.1415926e2 // 3.1415926 乘以 10的二次方

    # 复数

    复数有两种：complex64和complex128，分别表示其实部和虚部为32位和64位

    var c1 complex64
c1 = 1 + 2i
var c2 complex128
c2 = 2 + 3i
fmt.Println(c1, c2)

c3 := complex(3.14, 12)
// 得到实部和虚部
fmt.Println(real(c3), imag(c3)) // 实部，虚部

    # 布尔型

    go中使用bool进行声明数据，且布尔型数据只有true和false两个值

    1. go中不允许将整型强转为布尔型
    2. 布尔型无法参与数值运算，也无法和其他类型进行转换

    # 字符串

    go中字符串是以基本数据类型出现的，其内部使用UTF-8编码，使用双引号"包裹

    s1 := "hello go语言"
    1

    # 转义符

    转义符 含义
    \r 回车符(返回到行首)
    \n 换行符(直接跳到下一行的同列位置)
    \t 制表符
    \' 单引号
    \" 双引号
    \\ 反斜杠

    # 多行字符串

    使用反引号 `

    s1 := `one
	two
	three
	`
fmt.Println(s1)
    1
    2
    3
    4
    5

    反引号间换行将被做完字符串中的换行，但是所有的转义字符均无效，文本会原样输出

    # 字符串常用操作

    方法 介绍
    len(str) 求长度
    s1 + s2 或者 fmt.Sprintf(format string, a ...interface{}) string 拼接字符串
    strings.Split(s string, sep string) []string 分割字符串
    strings.Contains(s string, substr string) bool 判断是否包含目标串
    strings.HasPrefix(s string, prefix string) bool 判断前缀
    strings.HasSuffix(s string, suffix string) bool 判断后缀
    strings.Index(s string, substr string) int 从前往后找子串出现的位置
    strings.LastIndex(s string, substr string) int 从后往前找子串出现的位置
    strings.Join(elems []string, sep string) string 合并字符串数组

    # byte和rune类型（字符类型）

    在go中，字符使用单引号'包裹，可以是一个字母、汉字、符号

    var a := '中'
    1

    在go中字符有两种：

    1. uint8，别名为byte类型，代表ASCII码的一个字符
    2. int32，别名为rune类型，代表一个UTF-8字符，当需要处理中文、韩文或者其他复合字符时候使用

    # 修改字符串

    如果要修改字符串，需要先将其转成rune数组[]rune或者byte数组[]byte，改完后再转回string。无论哪种转换，都会重新分配内存，并复制字节数组。

    s1 := "hello"
b1 := []byte(s1)
b1[0] = 'a'
fmt.Println(string(b1))

    # 类型转换

    在go中，只有强转没有隐转，且只能在两个类型之间支持互相转换的时候使用

    基本语法 T(表达式) T表示要转换的类型，表达式可以是变量、复杂算子、和函数返回值

    var n1 int = 10
var f1 float64
f1 = float64(n1)
fmt.Println(f1)
