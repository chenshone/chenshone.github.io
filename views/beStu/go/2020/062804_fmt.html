    go学习(十一) fmt标准库

    CHENSHONE 2020/6/28 golang

    fmt包实现了类似C语言的printf和scanf的格式化I/O。

    # 输出

    # Print

    函数 效果
    Print(a ...interface{}) (n int, err error) 直接输出内容
    Printf(format string, a ...interface{}) (n int, err error) 支持格式化输出字符串
    Println(a ...interface{}) (n int, err error) 输出内容后换行

    # 输入

    # Scan

    函数 效果
    Scan(a ...interface{}) (n int, err error) 输入扫描文本，读取由空白符分隔的值保存到传递给本函数的参数中，换行符视为空白符
    Scanf(format string, a ...interface{}) (n int, err error) 支持格式化
    Scanln(a ...interface{}) (n int, err error) 扫描的换行符时结束

    输入和C语言一样，需要使用取地址符 &

    # bufio.NewReader

    有时候我们想完整获取输入的内容，而输入的内容可能包含空格，这种情况下可以使用 bufio 包来实现。

    func bufioDemo() {
	reader := bufio.NewReader(os.Stdin) // 从标准输入生成读对象
	fmt.Print("请输入内容：")
	text, _ := reader.ReadString('\n') // 读到换行
	text = strings.TrimSpace(text)
	fmt.Printf("%#v\n", text)
}

    # Fscan系列

    这几个函数功能分别类似于fmt.Scan、fmt.Scanf、fmt.Scanln三个函数，只不过它们不是从标准输入中读取数据而是从io.Reader中读取数据。

    func Fscan(r io.Reader, a ...interface{}) (n int, err error)
func Fscanln(r io.Reader, a ...interface{}) (n int, err error)
func Fscanf(r io.Reader, format string, a ...interface{}) (n int, err error)

    # Sscan系列

    这几个函数功能分别类似于fmt.Scan、fmt.Scanf、fmt.Scanln三个函数，只不过它们不是从标准输入中读取数据而是从指定字符串中读取数据。

    func Sscan(str string, a ...interface{}) (n int, err error)
func Sscanln(str string, a ...interface{}) (n int, err error)

    # 格式化占位符

    # 通用占位符

    占位符 说明
    %v 值的默认格式
    %+v 在输出结构体时会添加字段名
    %#v 值的go语法表示
    %T 输出值的类型
    %% 百分号

    # 布尔型

    占位符 说明
    %t true或false

    # 整型

    占位符 说明
    %b 值对应的二进制
    %c 值对应的Unicode码
    %d 值对应的十进制
    %o 值对应的八进制
    %x 值对应的十六进制，使用a-f
    %X 值对应的十六进制，使用A-F
    %U Unicode格式
    %q 值对应的go语法字符字面值

    # 浮点型

    占位符 说明
    %b 无小数部分、二进制指数的科学计数法
    %e 科学计数法，使用e
    %E 科学计数法，使用E
    %f 有小数部分但无指数部分
    %F 同%f
    %g 根据实际情况采用%e或%f格式
    %G 根据实际情况采用%E或%F格式

    # 字符串和[]byte

    占位符 说明
    %s 直接输出字符串或者[]byte
    %q 值对应的go语法字符串字面值
    %x 每个字节用两字符十六进制数表示，使用a-f
    %X 每个字节用两字符十六进制数表示，使用A-F

    # 指针

    占位符 说明
    %p 表示为十六进制，并加上前面的ox

    # 宽度、精度标识符

    占位符 说明
    %f 默认宽度，默认精度
    %9f 宽度为9，默认精度
    %.2f 默认宽度，精度为2
    %9.2f 宽度为9，精度为2
    %9.f 宽度为9，精度为0

    # 其他占位符

    占位符 说明
    '+' 输出数组的正负号
    ' ' 对数值，正数前加空格，负数前加负号；对字符串则各个字节间加空格
    '-' 输出左对齐
    '#' 添加格式，如八进制前加o，十六进制前加ox
    'o' 使用o而不是空格填充，对应数组类型会把填充的o放在正负号后面

    # Sprint

    Sprint系列函数会把传入的数据生成并返回一个字符串

    函数 效果
    Sprint(a ...interface{}) string 使用默认格式为其操作数设置Sprint格式，并返回结果字符串。当都不是字符串时，在操作数之间添加空格。
    Sprintf(format string, a ...interface{}) string 根据格式说明符设置格式，并返回结果字符串。
    Sprintln(a ...interface{}) string 使用其操作数的默认格式来格式化并返回结果字符串。
