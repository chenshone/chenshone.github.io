    # 切片

    因为数组的长度是固定的，所以数组有很多局限性。

    切片(slice)是一个拥有相同类型元素的可变长度的序列。它是基于数组类型做了一层封装。支持自动扩容。切片是一个引用类型，它的内部结构包含地址、长度和容量。

    # 定义

    // 定义一个int类型的切片
var a1 []int

// 没有初始化时，为空(nil)
fmt.Println(a1 == nil) // True

    # 长度和容量

    因为切片是数组的一层包装，底层还是数组，只不过是其他开好一些内存空间以供使用，容量就是底层开辟的数组空间，空间不够时就会扩容。长度就是里面元素的个数。

    长度使用len()获取，容量使用cap()获取

    # 基于数组定义切片

    数组定义切片是左闭右开

    a1 := [5]int{0,1,2,3,4}
a2 := a1[1:4]
a3 := a1[1:]
a4 := a1[:4]
a5 := a1[:]
fmt.Println(a2) // [1 2 3]
fmt.Println(a3) // [1 2 3 4]
fmt.Println(a4) // [0 1 2 3]
fmt.Println(a5) // [0 1 2 3 4]

    # 切片再切片

    切片也可以被切片

    a1 := [5]int{0,1,2,3,4}
a2 := a1[1:3]
a3 := a2[1:4]
fmt.Println(a2) // [1 2]
fmt.Println(a3) // [2 3 4]

    2
    3

    对切片再切片时，索引不能超过原数组的长度。

    这里不能超过原数组长度而不是切片长度，是因为切片是引用类型，它所存的是该切片的第一个元素所对应的底层数组的内存地址。及时切片索引超过被切片的长度也无妨，只要不超过原数组的长度即可

    # 使用make()函数构造切片

    如果想要不通过数组而是动态创建一个切片，可以使用make()函数

    /**
T: 切片中的元素类型
size: 元素数量
cap?: 容量,如果没写，就是和size一样
*/
make([]T, size, cap?)
