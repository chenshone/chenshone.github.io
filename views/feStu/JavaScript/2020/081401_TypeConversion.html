    # 字符串和数字之间的隐式强制类型转换

    + 运算符即能用于数字加法，也能用于字符串拼接。

    let a = 1
let b = a + ''
console.log(typeof b) // string

    # 隐式强制类型转换为布尔值

    1. if (..) 语句中的条件判断表达式
    2. for ( .. ; .. ; .. ) 语句中的条件判断表达式（第二个）

    # 宽松相等和严格相等

    简单的说，== 允许在相等比较中进行强制类型转换，而 === 不允许。

    # 抽象相等

    抽象相等规定了 == 的行为

    1. 如果两个值的类型相等，就直接比较值是否相等

    有两个情况特殊，需要注意：

    • NaN 不等于 NaN
    • +0 等于 -0
    1. 如果比较是两个对象(包括函数和数组)，那么如果两个对象指向同一个值时即视为相等，不发生强制类型转换
    2. 如果比较的一个是数字，一个是字符串，则会将字符串转成数字再比较
    3. 如果比较的其中一个元素为布尔值，而另一个不是，则会将布尔值转成数组再比较
    4. 对于 null 和 undefined，二者互相比较是相等的，与其他值比较均为false
    5. 如果是对象与非对象之间比较，则会将对象隐式转成非对象再比较

    题目：
如何让下面的条件成立
if (a == 2 && a == 3) {
  console.log("Yep, this happened.")
}

    解答：

    let i = 2
Number.prototype.valueOf = function () {
  return i++
}

let a = new Number(233)

if (a == 2 && a == 3) {
  console.log("Yep, this happened.")
}
