    KMP

    CHENSHONE 2020/11/15 算法思维

    # 问题背景 字符串匹配问题

    一个字符串str和一个字符串match 判断match是否为str的一个子串 如果是，返回第一个匹配的子串的第一个字符在str中的位置 如果不是，返回-1

    # 暴力解

    str从头开始遍历，下标记为i，每次都以位置的字符开头，遍历match字符，进行比较

    如果match字符全部符合，就返回第一字符在str中的位置

    如果有字符不符合，break，然后到下一个i位置，重复上诉操作

    如果记str长度为N，match长度为M，那么时间复杂度为O(N*M)

    # KMP

    KMP算法也是从左向右遍历的匹配，只不过是在遍历的过程中有加速，会跳过一些位置。

    # 前置概念

    想要做到遍历过程中跳过某些位置，就必须记录某些东西：

    # 一个字符串中每一个位置的之前字符串的前缀和后缀的最大匹配长度

    举个例子：

    str:a b c a b c d

    下标6的字符d，其之前的字符串为abcabc

    注意

    前缀和后缀不要取到字符串整体

    前缀 长度 1 a, 后缀 长度 1 c -> ❌

    前缀 长度 2 ab, 后缀 长度 2 bc -> ❌

    前缀 长度 3 abc, 后缀 长度 3 abc -> ⭕

    前缀 长度 4 abca, 后缀 长度 4 cabc -> ❌

    前缀 长度 5 abcab, 后缀 长度 5 bcabc -> ❌

    那么，下标为6的位置，其之前字符串的前缀和后缀的最大匹配长度为3

    # next数组

    对match字符串的每个字符都求其之前字符串的前缀和后缀的最大匹配长度，所构成的数组，称之为next数组

    next数组的0位置为-1，1位置为0

    # 关键算法流程

    和暴力算法一样，也是str和match字符挨个比对

    在某个时刻，str在x位置，match在y位置，他俩不等了

    那么，此时查找next数组中y位置的值为m，将y位置跳转到m位置，x在原位置不变

    然后x和y继续向后匹配

    # 解析

    之所以能y跳到m位置，x停在原位置，是因为

    假设，当前比较是从str的i位置、match的0位置开始匹配，直到i到x位置，match匹配的y位置时候，他俩不等了

    那么str前面的i ~ x-1位置是和match中0 ~ y - 1位置相匹配的

    此时查找next数组中y位置的值，根据next数组的定义，该值表示y位置之前的字符串的前缀与后缀的最大匹配长度，记为m

    因为str前面的i ~ x - 1位置是和match中0 ~ y - 1位置相等的，且y位置之前的字符串的前缀与后缀的最大匹配长度为m

    那么，str的x - m ~ x - 1和match的0 ~ m - 1位置是相匹配的, 相当于下一个匹配的是从str的x - m位置开始的

    所以，可以从str的x位置和match的m位置开始继续匹配

    隐藏条件

    上述操作中，存在一个隐含条件，那就是str的i位置开始匹配的情况下，在str的x位置，match的y位置不匹配，且next[y]的值为m，那么下一个可能匹配的位置就是str的x-m位置开始，记这个位置为j

    那么，在i和j中间，是否真的就不存在可能匹配的位置开头呢 ？

    通过反证法，假设在i和j中间，存在可能匹配的位置开头，记为k

    那么首先，k ~ x - 1这段必然与match的开头的一段匹配，这显然是和next[y]的值是矛盾的

    因为，如果k ~ x - 1这段与match的开头的一段匹配，且 k < j,那么，x - 1 - k 必然大于next[y]的m，而next数组的定义是前缀与后缀的最大匹配长度，这显然是矛盾的，所以str的i位置开始匹配的情况下，在str的x位置，match的y位置不匹配，且next[y]的值为m，那么下一个可能匹配的位置就是str的x-m位置开始

    # 代码

    public static int getIndexOf(String s, String m) {
    if (s == null || m == null || m.length() < 1 || s.length() < m.length()) {
        return -1;
    }
    char[] str = s.toCharArray();
    char[] match = m.toCharArray();
    int x = 0; // str中当前比对的位置
    int y = 0; // match中当前比对的位置
    int[] next = getNextArray(match); // 得到next数组的方法
    while (x < str.length && y < match.length) {
        if (str[x] == match[y]) {
            x++;
            y++;
        } else if (next[y] == -1) { // 只有next[0]位置为-1，此时y位置已经是0了
            x++;
        } else {
            y = next[y];
        }
    }
    return y == match.length ? x - y : -1;
}

    # 复杂度

    KMP算法可以做到O(N)的时间复杂度，显然上面的代码中，两个toCharArray都是O(N)的，下面while循环也是O(N)的，那么next数组的生成，做到O(M)的时间复杂度就可以了。

    # next数组的生成

    要做到O(M)的复杂度，就意味着遍历一遍match即可得到next数组，那么就需要重复利用已经得到的next[之前位置]的值

    假设，当前计算到i位置，i-1位置已经得到了，记next[i-1]为m，也就是0 ~ m - 1和i - 2 - m ~ i - 2是相同的

    1. m位置和i - 1位置相同 -> 那么next[i]就是m+1
    2. m位置和i - 1位置不相同 -> 那么将 i - 1位置和next[m]重复进行上述操作，直到0位置和i - 1位置不相等时，next[i] = 0，i++

    # 代码

    private static int[] getNextArray(char[] match) {
    if (match.length == 1) {
        return new int[]{-1};
    }
    int[] next = new int[match.length];
    next[0] = -1;
    int i = 2;
    // cn表示 cn位置的字符是当前和i-1位置比较的字符
    int cn = 0;
    while (i < next.length) {
        if (match[i - 1] == match[cn]) {
            next[i++] = ++cn;
        } else if (cn > 0) {
            cn = next[cn];
        } else {
            next[i++] = 0;
        }
    }
    return next;
}

    # 例题一

    给定两个树a、b,判断a中是否有某个子树的结构，和以b为头的树，完全一样的

    # 暴力解

    // 判断big中是否有某个子树的结构，和以small为头的树，完全一样的
public static boolean containsTree1(Node big, Node small) {
    if (small == null) {
        return true;
    }

    if (big == null) {
        return false;
    }

    if (isSameValueStructure(big, small)) {
        return true;
    }

    return containsTree1(big.left, small) && containsTree1(big.right, small);
}

private static boolean isSameValueStructure(Node head1, Node head2) {
    if (head1 == null && head2 == null) {
        return true;
    }
    if (head1 == null || head2 == null) {
        return false;
    }

    if (head1.value != head2.value) {
        return false;
    }

    return isSameValueStructure(head1.left, head2.left) && isSameValueStructure(head1.right, head2.right);
}

    # kmp

    将两个树都按照先序的方法序列化成一个数组，然后用kmp，判断b数组是否为a数组的一个子串,如果是，就返回true

    public static boolean containsTree2(Node big, Node small) {
    if (small == null) {
        return true;
    }

    if (big == null) {
        return false;
    }

    ArrayList<String> b = preSerial(big);
    ArrayList<String> s = preSerial(small);
    String[] str = new String[b.size()];
    for (int i = 0; i < str.length; i++) {
        str[i] = b.get(i);
    }

    String[] match = new String[b.size()];
    for (int i = 0; i < match.length; i++) {
        match[i] = s.get(i);
    }

    return getIndexOf(str, match) != -1;
}

private static ArrayList<String> preSerial(Node head) {
    ArrayList<String> ans = new ArrayList<>();
    pres(head, ans);
    return ans;
}

private static void pres(Node head, ArrayList<String> ans) {
    if (head == null) {
        ans.add(null);
    } else {
        ans.add(String.valueOf(head.value));
        pres(head.left, ans);
        pres(head.right, ans);
    }
}

private static int getIndexOf(String[] str, String[] match) {
    if (str == null || match == null || str.length < 1 || str.length < match.length) {
        return -1;
    }
    int x = 0;
    int y = 0;
    int[] next = getNextArray(match);

    while (x < str.length && y < match.length) {
        if (isEqual(str[x], match[y])) {
            x++;
            y++;
        } else if (next[y] == -1) {
            x++;
        } else {
            y = next[y];
        }
    }

    return y == match.length ? x - y : -1;
}

private static boolean isEqual(String a, String b) {
    if (a == null && b == null) {
        return true;
    }
    if (a == null || b == null) {
        return false;
    }
    return a.equals(b);
}

private static int[] getNextArray(String[] match) {
    if (match.length == 1) {
        return new int[]{-1};
    }

    int[] ans = new int[match.length];
    ans[0] = -1;
    int pre = 0;
    int i = 2;
    while (i < match.length) {
        if (isEqual(match[i - 1], match[pre])) {
            ans[i++] = ++pre;
        } else if (pre > 0) {
            pre = ans[pre];
        } else {
            ans[i++] = 0;
        }
    }
    return ans;
}
