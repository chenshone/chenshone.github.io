    Manacher算法

    CHENSHONE 2020/11/18 算法思维

    回文串

    回文串就是一个字符串，正着念和反着念是一样的

    Manacher算法也叫马拉车算法，是解决求一个字符串中最长回文子串的长度。

    # 暴力解

    # 思路

    从左向右依次遍历，以每个元素为中间位置，向两边扩，直到扩到不能扩为止，例如“ab12321cd”,可以找到最长回文子串是“12321”

    但是这种找法存在一个问题，只能找到奇数长度的回文串，对于偶数长度的，例如，“ab1221cd”就无能为力了，因为流程是找一个元素作为可能子串的中间位置，如果是偶数长度，这个中间位置是没有的。

    所以，可以将字符串改造一下，将每个字符之间以及头和尾都插入一个标记字符“#”,例如，“ab1221cd” -> “#a#b#1#2#2#1#c#d#”

    一旦这样做之后，就可以得到偶数长度的可能回文串了

    # Manacher算法

    Manacher算法可以做到O(N)时间内完成该任务

    # 一些概念

    1. 回文半径：从回文中心往一侧数，数到最边边，的数量，“f#1#a#1#s” -> 4
    2. 回文半径数组pArr[]：记录以每个位置为中间位置的最长回文串的半径
    3. 回文最右边界（int R)：当前最长回文串的最右右边界，只要新的回文串的最右右边界比现在的靠右，就更新，
    4. 中心（int C）：上面回文最右边界对应的那个回文串的中心位置，只要上面的回文最右边界更新，中心就更新成最新的

    # 算法流程

    从左向右遍历结构化过的字符串，假设当前节点为i位置

    1. i位置在R的右边（在R外）：跟暴力过程一样，依次往两边扩，直到不能扩位置
    2. i位置在R的左边或者就在R位置（在R内）：如果i在R内，那么C必然在i的左边，因为C是以R为最右边界的回文串的中间位置，而这个回文串又是在遍历到i之前就得到的，那么C必然在i的左边。既然这样，就能得到以C为中心点，R为右边界的回文串，也能得到对应的左边界L，以及以C为对称点的i'。而且我们还能根据pArr[]拿到i'的回文半径，也就难道了以i'为中心的最长回文串，这个回文串有如下几种情况
      1. 这个回文串全在L - R内部：因为L - R这个回文串是对称的，所以i位置的最长回文串也就跟i'位置一样。
      2. 这个回文串有部分在L - R外部：这种情况也i不需要往外扩，因为如果需要往外扩，那么L - R这个长度的回文串也会相应外扩，而L - R已经是确定的了，这是矛盾的，所以i位置的最长回文半径也就是R - i
      3. 这个回文串的左边界是L - R的左边界（L位置）：那么i位置为中心的串，起码到R位置是回文的，R之外，不确定，需要挨个确定。

    # 代码

    public static int manacher(String str) {
    if (str == null || str.length() == 0) {
        return 0;
    }
    // "12321" -> "#1#2#3#2#1#"
    char[] s = manacherString(str);
    // 回文半径数组
    int[] pArr = new int[s.length];
    int C = -1;
    int R = -1; // 最右边界的后一个位置，也就是前面最长回文串第一个匹配失败的位置
    int max = Integer.MIN_VALUE;
    for (int i = 0; i < s.length; i++) {
        // 求i为中心位置至少的半径
        // i >= R 表示i在R外，起码i本身一个字符是回文的
        pArr[i] = R > i ? Math.min(pArr[2 * C - i], R - i) : 1;
        while (i + pArr[i] < s.length && i - pArr[i] > -1) { // 左右两侧不越界的情况下
            if (s[i + pArr[i]] == s[i - pArr[i]]) {
                pArr[i]++;
            } else {
                break;
            }
        }
        if (i + pArr[i] > R) {
            R = i + pArr[i];
            C = i;
        }
        max = Math.max(max, pArr[i]);
    }
    return max - 1;
}

private static char[] manacherString(String str) {
    char[] charArr = str.toCharArray();
    char[] res = new char[str.length() * 2 + 1];
    int index = 0;
    for (int i = 0; i != res.length; i++) {
        res[i] = (i & 1) == 0 ? '#' : charArr[index++];
    }
    return res;
}

    # 应用

    给定一个字符串，在其后面添加字符，使其成为回文串，试问，最少添加多少个字符可以完成

    # 思路

    就是求包含原字符串最右一个位置的最长子回文串（也就是包含最右位置的回文串的最左边的那个回文中心），找到这个串后，将左侧不在范围内的字符反过来放到右边，整个字符串就是回文串了。
