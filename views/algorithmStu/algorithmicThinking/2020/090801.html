    排序算法（一）选择、冒泡、插入

    CHENSHONE 2020/9/8 sort算法思维

    不作任何说明的话，排序都按升序来。

    # 选择排序 不稳定

    # 思路

    选择排序，简单的说就是每次遍历都将这一遍遍历中都最小值挑出来放到最前面（已经排好序的那部分）

    # 代码

    public class SelectionSort {
    public static void selectionSort(int[] arr) {
        if (arr == null || arr.length < 2) {
            return;
        }
        for (int i = 0; i < arr.length - 1; i++) {
            int minIndex = i;
            for (int j = i + 1; j < arr.length; j++) {
                minIndex = arr[j] < arr[minIndex] ? j : minIndex;
            }
            swap(arr, i, minIndex);
        }
    }

    public static void swap(int[] arr, int i, int j) {
        int tmp = arr[i];
        arr[i] = arr[j];
        arr[j] = tmp;
    }
}

    # 冒泡排序 稳定

    # 思路

    冒泡排序，顾名思义，跟冒泡泡是很像的，遍历到一个数的时候，将他和后面的一个数比较，如果比后面的那个大，就swap位置，然后向后遍历，这样一来，每遍历一次，最后面的就是当前遍历的这组中最大的。

    # 代码

    public class BubbleSort {
    public static void bubbleSort(int[] arr) {
        if (arr == null || arr.length < 2) {
            return;
        }

        for (int e = arr.length - 1; e > 0 ; e--) {
            for(int i = 0; i < e; i++) {
                if (arr[i] > arr[i + 1]) {
                    swap(arr, i, i+ 1);
                }
            }
        }
    }

    public static void swap(int[] arr, int i, int j) {
        int tmp = arr[i];
        arr[i] = arr[j];
        arr[j] = tmp;
    }
}

    # 插入排序 稳定

    # 思路

    插入排序就是说，从左往右依次遍历每个数，对于这个数，将它与它之前的每个数从右往左比较，如果比前面的数小，就swap，直到比前面的数大或者没有数停止，继续遍历下一个数。

    # 代码

    public class InsertionSort {
    public static void insertionSort(int[] arr) {
        if (arr == null || arr.length < 2) {
            return;
        }

        for (int i = 1; i < arr.length; i++) {
            for (int j = i - 1; j >= 0 && arr[j] > arr[j + 1]; j--) {
                swap(arr, j, j + 1);
            }
        }
    }

    public static void swap(int[] arr, int i, int j) {
        // arr[i] = arr[i] ^ arr[j];
        // arr[j] = arr[i] ^ arr[j];
        // arr[i] = arr[i] ^ arr[j];
        int tmp = arr[i];
        arr[i] = arr[j];
        arr[j] = tmp;
    }
}
