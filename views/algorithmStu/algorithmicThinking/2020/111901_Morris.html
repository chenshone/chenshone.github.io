    遍历二叉树的方法，不管是递归还是迭代，时间复杂度都是O(N)，额外空间复杂度都是O(h)，也就是整个数的高度，Morris遍历可以做到时间复杂度O(N)，额外空间复杂度O(1)。

    # 流程

    假设当前节点为cur，一开始cur来到整棵树的根节点，下面的步骤知道cur == null的时候停止

    1. cur无左树，cur = cur.right
    2. cur有左树，找到左树的前驱节点mostRight
      1. mostRight的右指针指向null, mostRight.right = cur, cur = cur.left
      2. mostRight的右指针指向cur, mostRight.right = null, cur = cur.right

    如果按照这个流程走，有左孩子的节点，会经过2次，没有左孩子的节点，只会经过1次。这就是morris序。

    # 代码

    public static void morris(Node head) {
    if (head == null) {
        return;
    }
    Node cur = head;
    Node mostRight = null;
    while (cur != null) {
        mostRight = cur.left;
        if (mostRight != null) { // 如果有前驱节点
            while (mostRight.right != null && mostRight.right != cur) {
                mostRight = mostRight.right;
            }
            if (mostRight.right == null) { // 第一次进
                mostRight.right = cur;
                cur = cur.left;
                continue;
            } else { // mostRight.right -> cur ， 第二次进
                mostRight.right = null;
            }
        }
        cur = cur.right;
    }
}

    # 中序遍历

    在第一次经过该节点时打印，只经过一次的，经过就打印

    public static void morrisIn(Node head) {
    if (head == null) {
        return;
    }
    Node cur = head;
    Node mostRight = null;
    while (cur != null) {
        mostRight = cur.left;
        if (mostRight != null) {
            while (mostRight.right != null && mostRight.right != cur) {
                mostRight = mostRight.right;
            }
            if (mostRight.right == null) {
                mostRight.right = cur;
                cur = cur.left;
                continue;
            } else { // mostRight.right -> cur
                mostRight.right = null;
            }
        }
        // 每个节点都会往右走，经过两次的节点，第一次是往左走，所以只要往右走，就是第二次经过该节点，或者这个节点只经过一次
        // 中序遍历处理cur节点位置
        System.out.println(cur.value + " ");
        cur = cur.right;
    }
    System.out.println();
}

    # 先序遍历

    在第二次经过该节点时打印，只经过一次的，经过就打印

    public static void morrisPre(Node head) {
    if (head == null) {
        return;
    }
    Node cur = head;
    Node mostRight = null;
    while (cur != null) {
        mostRight = cur.left;
        if (mostRight != null) {
            while (mostRight.right != null && mostRight.right != cur) {
                mostRight = mostRight.right;
            }
            if (mostRight.right == null) {
                mostRight.right = cur;

                // 这个节点有左树才会进，而且前驱节点的右节点为null，就意味着是第一次经过该节点
                // 先序遍历处理cur节点位置
                System.out.println(cur.value + " ");

                cur = cur.left;
                continue;
            } else { // mostRight.right -> cur
                mostRight.right = null;
            }
        } else { // 这边是没有左树的节点，只经过一次 , 先序遍历处理cur节点位置
            System.out.println(cur.value + " ");
        }
        cur = cur.right;
    }
    System.out.println();
}

    # 后序遍历

    当cur第二次经过某一个点时候，逆序处理其左树的右边界（左树的从根节点开始，各自的右孩子），处理完所有经过两次的节点后，逆序处理整棵树的右边界。

     public static void morrisPos(Node head) {
    if (head == null) {
        return;
    }
    Node cur = head;
    Node mostRight = null;
    while (cur != null) {
        mostRight = cur.left;
        if (mostRight != null) {
            while (mostRight.right != null && mostRight.right != cur) {
                mostRight = mostRight.right;
            }
            if (mostRight.right == null) {
                mostRight.right = cur;
                cur = cur.left;
                continue;
            } else { // mostRight.right -> cur
                // 第二次经过该节点时候，逆序处理其左树的右边界
                mostRight.right = null;
                printEdge(cur.left);
            }
        }
        cur = cur.right;
    }
    // 最后逆序处理整棵树的右边界
    printEdge(head);
    System.out.println();
}

private static void printEdge(Node head) {
    Node tail = reverseEdge(head);
    Node cur = tail;
    while (cur != null) {
        // 处理节点
        System.out.println(cur.value + " ");
        cur = cur.right;
    }
    reverseEdge(tail);
}

private static Node reverseEdge(Node head) {
    Node pre = null;
    Node next = null;
    while (head != null) {
        next = head.right;
        head.right = pre;
        pre = head;
        head = next;
    }
    return pre;
}

    # 判断是否为BST（搜索二叉树）

    public static boolean isBST(Node head) {
    if (head == null) {
        return true;
    }
    Node cur = head;
    Node mostRight = null;

    Integer pre = null;
    while (cur != null) {
        mostRight = cur.left;
        if (mostRight != null) {
            while (mostRight.right != null && mostRight.right != cur) {
                mostRight = mostRight.right;
            }
            if (mostRight.right == null) {
                mostRight.right = cur;
                cur = cur.left;
                continue;
            } else { // mostRight.right -> cur
                mostRight.right = null;
            }
        }
        // 每个节点都会往右走，经过两次的节点，第一次是往左走，所以只要往右走，就是第二次经过该节点，或者这个节点只经过一次
        // 中序遍历处理cur节点位置
        if (pre != null && pre >= cur.value) {
            return false;
        }
        pre = cur.value;
        cur = cur.right;
    }
    return true;
}

    # 求二叉树的最小高度

    public static int minHeight(Node head) {
    if (head == null) {
        return 0;
    }
    Node cur = head;
    Node mostRight = null;
    int curLevel = 0;
    int minHeight = Integer.MAX_VALUE;
    while (cur != null) {
        mostRight = cur.left;
        if (mostRight != null) {
            int rightBoardSize = 1; // 左树右边界的数量
            while (mostRight.right != null && mostRight.right != cur) {
                rightBoardSize++;
                mostRight = mostRight.right;
            }
            if (mostRight.right == null) { // 第一次到达
                curLevel++;
                mostRight.right = cur;
                cur = cur.left;
                continue;
            } else { // 第二次到达
                if (mostRight.left == null) { // 判断这个前驱节点是否为叶节点，是就比较
                    minHeight = Math.min(minHeight, curLevel);
                }
                curLevel -= rightBoardSize; // 恢复cur节点的高度
                mostRight.right = null;
            }
        } else { // 只有一次到达
            curLevel++;
        }
        cur = cur.right;
    }

    // 整棵树的最右节点上面的过程管不到，需要单独判断
    int finalRight = 1;
    cur = head;
    while (cur.right != null) {
        finalRight++;
        cur = cur.right;
    }
    if (cur.left == null) { // 判断最右节点是否为叶节点
        minHeight = Math.min(minHeight, finalRight);
    }
    return minHeight;
}

    # 适用范围

    如果需要左孩子的信息和右孩子的信息，那么不能使用morris遍历。

    因为在morris遍历中，我们只能根据是否当前节点是否是第二次经过（如果有前驱节点的话），这意味着，如果做信息记录的话，我们只能拿到一侧的数据。
