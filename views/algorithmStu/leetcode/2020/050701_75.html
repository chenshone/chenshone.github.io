    # 👀题目 (opens new window)

    题目内容

    给定一个包含红色、白色和蓝色，一共 n 个元素的数组，原地对它们进行排序，使得相同颜色的元素相邻，并按照红色、白色、蓝色顺序排列。

    此题中，我们使用整数 0、 1 和 2 分别表示红色、白色和蓝色。

    注意

    不能使用代码库中的排序函数来解决这道题。

    示例

    输入: [2,0,2,1,1,0]

    输出: [0,0,1,1,2,2]

    # 💡 思路

    因为只有三个数，我们要做的就是把 0 放最左边，2 放最右边，剩下的 1 自然就在最中间了。

    因为要有2种操作/比较(一个移到最右边，一个移到最左边)， 所以设置2个指针，分别指向数组的头和尾。还需要一个指针来遍历数组。

    # ✍️ 代码

    class Solution:
    def sortColors(self, nums: List[int]) -> None:
        """
        Do not return anything, modify nums in-place instead.
        """
        p = 0 # 遍历数组的指针
        left = 0 # 交换 0 的位置
        right = len(nums) - 1 # 交换 2 的位置
        while p <= right: # 显然当 p > right的时候 整个数组都排序好了

        """
        这里swap 2 的时候不能让p++，而 swap 0 的时候可以。
        原因很简单，
          因为p是从左往右遍历的，
            swap 0是swap p指针已经走过的位置，是已经排序好的，值可能是 0 或 1， 所有不需要再次比较。
          而
            swap 2 交换过来的数字也可能是2，这里就需要再次比较了。
        """
            if nums[p] == 0:
                nums[p], nums[left] = nums[left], nums[p]
                p += 1
                left += 1
            elif nums[p] == 2:
                nums[p], nums[right] = nums[right], nums[p]
                right -= 1
            else:
                p += 1
