    leetcode 88.合并两个有序数组

    CHENSHONE 2020/5/7 arraytwo pioneereasy

    # 👀题目 (opens new window)

    题目内容

    给定两个有序整数数组 nums1 和 nums2，将 nums2 合并到 nums1 中，使得 num1 成为一个有序数组

    说明

    初始化 nums1 和 nums2 的元素数量分别为 m 和 n。

    你可以假设 nums1 有足够的空间（空间大小大于或等于 m + n）来保存 nums2 中的元素。

    示例

    输入:

    nums1 = [1,2,3,0,0,0], m = 3

    nums2 = [2,5,6], n = 3

    输出:

    [1,2,2,3,5,6]

    # 💡 思路

    首先nums1空间足够大，就需要手动扩容了。

    本题思路还是三指针，两个指针p1, p2分别指向nums1和nums2的尾部，还有一个cur指针指向nums1+nums2的尾部，p1和p2比较，大的放在cur位置，依次往前推进，直到nums2全部加到nums1为止

    # ✍️ 代码

    class Solution:
    def merge(self, nums1: List[int], m: int, nums2: List[int],
              n: int) -> None:
        p1 = m - 1
        p2 = n - 1
        cur = m + n - 1

        while p2 > -1:
            # 如果nums1全部放完了，那么就依次填入nums2的元素
            if p1 > -1 and nums1[p1] > nums2[p2]:
                nums1[cur] = nums1[p1]
                cur -= 1
                p1 -= 1
            else:
                nums1[cur] = nums2[p2]
                cur -= 1
                p2 -= 1
